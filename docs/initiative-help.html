What is an Initiative?

An Initiative (also called an Action Item) is a specific project or task that helps you achieve a key result or objective. While objectives define "what" you want to achieve and key results measure progress, initiatives define "how" you'll get there.

How to Create an Initiative

  1. Navigate to Initiatives section - Or add from within an objective/key result
  2. Click "Create Initiative" - Start defining your action item
  3. Enter initiative title - Be specific about the action (e.g., "Launch customer referral program")
  4. Add description - Explain the scope and expected outcome
  5. Link to key result/objective - Show which goal this initiative supports
  6. Assign owner - Who is responsible for execution?
  7. Set start and end dates - Define the timeline
  8. Add resources - List budget, team members, tools needed
  9. Click "Save" - Your initiative is ready to launch!

Best Practices

  • Keep initiatives focused - One clear action per initiative
  • Break down large initiatives - Create sub-tasks if needed
  • Assign clear ownership - One person accountable
  • Set realistic timelines - Include buffer for unexpected issues
  • Track dependencies - Note what must happen before/after
  • Regular status updates - Weekly check-ins on progress

Initiative vs. Key Result vs. Objective

Objective: Increase customer retention to 85%
Key Result: Reduce churn rate from 15% to 10%
Initiative: Implement customer success onboarding program

Common Questions

Q: How many initiatives should I have per key result?
A: Start with 2-4 initiatives. You can add more as you learn what works.

Q: Can an initiative support multiple key results?
A: Yes! Some strategic initiatives can impact multiple metrics.

Q: What if an initiative isn't working?
A: Don't be afraid to pause or cancel initiatives that aren't delivering results. Pivot quickly!
