What is a Key Result?

A Key Result is a measurable outcome that indicates progress toward an objective. Think of it as a specific, quantifiable milestone that helps you track whether you're achieving your objective.

How to Create a Key Result

  1. Select parent objective - Choose which objective this key result supports
  2. Click "Add Key Result" - Find this option within the objective details
  3. Write a clear description - Define what you're measuring (e.g., "Monthly active users")
  4. Set baseline value - What's the starting point? (e.g., 1,000 users)
  5. Set target value - What's your goal? (e.g., 5,000 users)
  6. Choose metric type - Percentage, number, currency, etc.
  7. Set deadline - When should this be achieved?
  8. Click "Save" - Your key result is now tracking!

Best Practices

  • Make them quantifiable - Use specific numbers, not vague descriptions
  • 3-5 key results per objective - More than 5 becomes hard to track
  • Set stretch goals - Aim for 70-80% achievement as "success"
  • Update regularly - Check-in weekly to track progress
  • Be outcome-focused - Measure results, not activities
üí° Pro Tip: Use leading indicators (predictive metrics) rather than lagging indicators (historical metrics) when possible!

Common Questions

Q: What's the difference between a key result and an initiative?
A: Key results measure outcomes ("what" you achieve). Initiatives are actions ("how" you achieve it).

Q: How often should I update key results?
A: Weekly check-ins are ideal for most key results.

Q: Can I have a key result without an objective?
A: No, key results must be linked to an objective to provide context.
